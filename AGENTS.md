# AGENTS.md

这个仓库是 multmatrix.com 的静态主页。

## 目标

把首页改成一个现代、漂亮、SEO 友好的 SaaS 产品落地页。

主页主要推广 Multmatrix 的 AI 字幕工具：

https://sub.multmatrix.com/

## 重要规则

- 保持静态网站，可以部署到 GitHub Pages / Cloudflare Pages。
- 优先修改现有 HTML/CSS。
- 不要引入 React、Next.js、Vue、Tailwind 或构建工具。
- 第一版不要安装任何依赖。
- 页面正文文案使用英文。
- Codex 可以用中文解释修改思路。
- 所有主要 CTA 都链接到 https://sub.multmatrix.com/
- 不要编造不存在的功能、价格、用户评价、合作伙伴。
- 页面要适配手机端。
- SEO 要自然，不要堆关键词。

## 页面应该包含

1. Header
2. Hero 首屏
3. 产品预览区域
4. 功能卡片
5. 使用流程
6. 使用场景
7. FAQ
8. Footer

## 视觉方向

- 现代 SaaS 风格
- 深色科技背景
- 蓝紫色渐变
- 卡片式布局
- 字体层级清晰
- 间距统一
- 移动端友好

## 完成前检查

- 检查所有 CTA 是否指向 https://sub.multmatrix.com/
- 检查是否只有一个 H1。
- 检查移动端布局。
- 不要加入无用依赖。
- 修改完成后说明改了哪些文件，以及如何本地预览。
